JavaScript中的数据类型

数据类型分类

js中的数据类型分为两大类,一种称为简单数据类型,另一种称为复杂数据类型.

简单数据类型也叫基本数据类型,其数据内容是固定的,存储在栈中,可以直接按值访问.

复杂数据类型页脚引用数据类型,其数据内容是会改变的,把数据存储在堆中,把数据的指针(链接)存储在栈中.

对于栈和堆的理解: 栈就相当于电脑中的内存条,堆相当于电脑中的硬盘.内存条(栈)的读取速度快但是容量小,只能存储一些简单数据,而硬盘(堆)的容量大,把复杂数据都存储在内,在内存条(栈)中创建一个快捷方式(指针)指向硬盘(堆)中的复杂数据.

基本数据类型

基本数据类型: String(字符串) / Number(数字) / Boolean(布尔) / Undefined(未定义) / Null(空).

String(字符串)必须用''或""包起来,可以使用单引号,也可以使用双引号,但是要保持一致不要混用.如果字符串中包含引号的时候,表示字符串的引号和字符串中的引号不能相同,如:var nick ='He is called "Tank".'或var nick ="He is called 'Tank'."

Number(数字)可以是整数或浮点数值，除此之外，Number 还有一个特殊的数值，即NaN (Not a Number)，NaN表示一个承诺返回数值类型数据但实际没有返回的情况。NaN有两个特点：一是任何涉及NaN的操作都会返回NaN，二是NaN不与任何数值相等，包括自己，但可以通过 isNaN() 方法来判断一个数值是否为 NaN .

typeof 123; //"number"
typeof 123.123; //"number"
typeof NaN; //"number"
NaN + 1; // NaN
NaN === NaN; //false
NaN === false; ///false
isNaN(NaN + 1); //true

Boolean(布尔)只有两个值:true或false.js中所有类型的值都可以转化成布尔值. 0 / NaN / "" / null / undefined都返回false,除此之外全部返回true.

typeof true; //"boolean"
typeof false; //"boolean"
Boolean(0); //false
Boolean(NaN); //false
Boolean(""); //false
Boolean(null); //false
Boolean(undefined); //false
Boolean(1); //true
Boolean("a"); //true
Boolean(new Function()); //true

Undefined(未定义)和Null(空)是两种不同的数据类型.undefined表示缺少值,就没有这个东西;而null表示值为空,是有这个东西的.

Number(undefined); //NaN
Number(null); //0

复杂数据类型

复杂数据类型: Object(对象),包括 Array(数组) / Function(函数) / Data(日期)等.

Object(对象)是属性和方法的集合,以键值对的形式存在,键和值以:(冒号)连接,属性和属性之间以,(逗号)分隔.js在解析的时候会把其中的键先转换成字符串再去读取对应的值.

因为复杂类型的数据是存储在堆中,把指针存储在栈中,所以变量之间的相互赋值实际上操作的是指针,而非数据本身.

var a = {x:1,y:2}; // var a = {'x':1,'y':2};
var b = a;

创建一个对象,通过变量a把指针存储在栈中,链接到堆中的对象(解析是会把键名x转换成'x').将变量a赋值给变量b只是把指针赋值给了变量b,并没有在堆中创建一个新对象.
