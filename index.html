js中的数据类型

js中数据类型分为两大类：基本数据类型和复杂数据类型。基本数据类型包括：Underfind（未定义）、Null（空）、Boolean（布尔）、Number（数字）、String（字符串），复杂数据类型包括：Array（数组）

文字超出部分省略号显示

为了保证页面的整洁美观，在很多的时候，我们常需要隐藏超出长度的文字。这在列表条目，题目，名称等地方常用到。可分为单行超出隐藏和多行超出隐藏两种方式

常用插件

components

图片懒加载

lazyload是一个实现图片延迟加载的jQuery 插件，它可以延迟加载长页面中的图片。在浏览器可视区域外的图片在初始状态下不会被载入，直到用户将页面滚动到它们所在的位置。

swiper轮播插件

Swiper 是一款免费以及轻量级的移动设备触控滑块的js框架，使用硬件加速过渡（如果该设备支持的话）。主要使用于移动端的网站、移动web apps，native apps和hybrid apps。

bootstrap组件模版

Bootstrap是Twitter推出的一个用于前端开发的开源工具包,它是最受欢迎的 HTML、CSS 和 JS 框架之一，用于开发响应式布局、移动设备优先的 WEB 项目

一键分享至多个社交平台

share.js是一个简单、重量轻、灵活、好看的jquery分享按钮提示插件。所有主要的社交网络有自己的jquery分享按钮，你可以将他们放在你的页面上。

css3前缀自动补齐

prefixfree.js是一个 JavaScript 插件，用于帮助你从 CSS 前缀的地狱中解脱出来。你编写的 CSS 代码无需填写浏览器前缀，在需要的时候，-prefix-free 会帮助你自动添加当前浏览器需要的前缀。

内容滚动到窗口区域显示

scrollReveal.js是一款页面滚动显示动画的JavaScript插件，能让页面更加有趣，更吸引用户眼球,动画可以播放一次或无限次.它的动画也是用 CSS3 创建的，所以它不支持 IE10 以下的浏览器。
